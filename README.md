# Cloud Bites

A short daily cloud study app for reviewing core AWS, infrastructure, DevOps, container, Kubernetes, Terraform, monitoring, and architecture concepts.

## Features

- Daily set of 5 cloud concepts
- Concept cards with summaries, analogies, practical use cases, and common confusion points
- OX and multiple-choice quizzes
- Wrong-answer review mode
- Local progress, streak, and review history using browser localStorage

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```

Open the local URL shown in the terminal.
