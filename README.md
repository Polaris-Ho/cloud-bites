# Cloud Bites

## 소개
이 프로젝트는 클라우드 인프라 학습을 짧은 개념 카드와 퀴즈로 반복할 수 있도록 만든 React 기반 학습 앱입니다.

AWS, 네트워크, 컴퓨팅, 스토리지, 데이터베이스, 보안, DevOps, 컨테이너, Kubernetes, Terraform, 모니터링, 아키텍처 개념을 하루 단위로 복습할 수 있도록 구성했습니다.

## 주요 기능
- 오늘 학습할 클라우드 개념 5개 제공
- 개념별 요약, 비유, 실무 활용, 혼동 포인트 정리
- OX 및 객관식 퀴즈를 통한 즉시 복습
- 틀린 문제를 따로 모아 다시 풀 수 있는 오답 복습 기능
- 학습 완료 수, 오답 수, 연속 학습일 기록
- 브라우저 localStorage 기반의 개인 학습 기록 저장
- 모바일과 데스크톱에서 사용할 수 있는 반응형 UI

## 사용 방법
1. 프로젝트 다운로드

```bash
git clone https://github.com/Polaris-Ho/cloud-bites.git
cd cloud-bites
```

2. 의존성 설치

```bash
npm install
```

3. 개발 서버 실행

```bash
npm run dev
```

4. 브라우저에서 로컬 주소 접속

```text
http://localhost:5173
```
